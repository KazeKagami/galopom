## getNextID
Универсальная функция получения следующего ID для любой модели.
**Параметры**
> ***model*** - mongoose модель (Attraction, User, Post и т.д.)
***idField*** - необязательное поле. Название поля ID (по умолчанию 'm_id')

**Описание**
Через методы mongoose проводит выборку столбца idField, с обратной сортировкой, получая последнюю запись. Возвращает последний id + 1.
Если же записей не найдено - возвращает 1.

**Синтаксис**
```js
// Если создалась новая коллекция:
getNextID(user) // 1
// Запись в базу:
getNextID(user) // id: 1 -> id: 2
```
