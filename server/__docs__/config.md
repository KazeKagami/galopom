# config
Содержит важные подключения к базам данных, другим системам.
***
## db
Подключение к базе данных
**Описание**
В зависимости от версии (dev / prod), берёт занчения для подключения к базе данных.
> **Важно!**\
> .env должен содержать 2 конфига: локалхост и подключение к удалённой базе

**Синтаксис**
```javascript
// app.js
const connectDB = require('./config/db');
connectDB();
```
Берёт ссылку на базу mongo из переменных окружения, используя dotenv.\
При успешном подключении - логирует и пропускает пользоваться.\
При неудачном подключении - выдаёт ошибку и прерывает попытку.
